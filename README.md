# MSnake

Real-time multiplayer Snake game!

## Run

```bash
pnpm install
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)

## Commands

- `pnpm build` — production build
- `pnpm test` — run tests

## Gameplay

- **Solo**: Steering via mouse pointer inside the board
- **Multiplayer**: Create room or join via invite link; steer with mouse. Optional: fire mode, time/score limits, self/snake collision
